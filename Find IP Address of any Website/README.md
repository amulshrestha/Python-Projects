# Find IP of any Website
This python program find IP address of any website which is given as input along with the host name and host IP.

## Dependencies/Module Requirements
- Python 3.6+
- socket

## Module Installation / Setup
- pip install socket

## Running The Program

```python findIP.py```
or
```python3 main.py```
